# What is type in TS?

## Used to create a type; keyword

A `Type` is a keyword that is used to creates a type alias in TS.

## Used to represent a variable's category

The TS contains primitive data types, data types that is introduced by TS, object type

## Primitive Types

Number - number
String - string
Boolean - boolean
Null - null
Undefined - undefined
Symbol - symbol
BigInt - bigint

## Object Types

Object
Array
Tuple
Enum
Function
Class
Interface

## Advanced Types

Union Types
Intersection Types
Literal Types
Mapped Types

## Type Assertion

The `as` keyword makes the type concrete by using the following type by `as`
